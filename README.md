# hul-common-lib
A common library for HUL based front-end electronics.
AMANEQ, CIRASAME, and RAYRAW software require this library.

## Tested system
Requires C++17.
| System | Version | Compiler                 | CMake                          |
| ---    | ---     | ---                      | ---                            | 
| AlmaLinux | 9       | GCC 11.3.1 (Require > 7.1) | 3.20.2 (Require > 3.8) |
